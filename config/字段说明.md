# 字段说明

```json
{
    "log": {
        "level": 2,         // 0: TRACE,   最精细的日志，可以用来追溯函数整个运行流程
                            // 1: DEBUG,   调试信息
                            // 2: INFO,    一般信息
                            // 3 WARN,     警告信息
                            // 4 ERROR,    错误信息
                            // 5 CRITICAL, 严重错误信息
                            // 6 OFF       关闭日志
        "size": 2,          // 单个日志文件大小,单位: MB
        "rotating": 5,      // 允许的最大日志文件数量，也是日志滚动数量
        "path": "/var/log"  // 日志存放的路径
    },
    "enable_bagger": false
}
```
